### 🎨 一、视觉风格指南（北美风格）
北美B2B工业企业网站的核心关键词：干净、自信、数据驱动、留白充足、无衬线字体、克制用色、强视觉冲击的实拍图。

1. 色彩体系
用途	色值示例	说明
主色	#0A2540 (深海军蓝)	代表专业、信任、技术深度
辅助色	#0066CC (科技蓝)	用于按钮、链接、强调数据
中性色	#F4F7FA (浅灰蓝背景)	区块分隔，干净
文字色	#1A1E24 (深灰) / #5A6874 (次要)	高可读性
强调/警示	#E85D04 (暖橙)	极少使用，仅用于核心CTA或数据亮点
2. 字体
标题：Inter / Roboto / Montserrat（无衬线，现代硬朗）

正文：Inter / Lato / Open Sans（极佳可读性）

等宽字体（参数表）：Fira Code / JetBrains Mono

3. 布局特征
宽屏全幅区块（1920px宽，内容居中1200px）

大量留白（区块间距80-120px）

网格/卡片式展示产品及应用

数字滚动动画（产能、市占率、良品率）

微交互：hover 卡片上浮、按钮填充、参数表行高亮

4. 图像与视频风格
工厂实拍：明亮、高锐度、人物不摆拍

石墨产品：微距结构、CNC加工火花、热场红热状态

不适用：过度修饰的3D渲染、无意义的抽象科技线条

视频：15-30秒产线延时摄影 + 核心设备特写
